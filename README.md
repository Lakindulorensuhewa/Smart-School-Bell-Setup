## 📌 Overview

Smart School Bell System is a Windows desktop application that replaces manual bell operators. It automatically rings scheduled audio files at precise times every day without human input.

| Layer | Technology |
|------|------------|
| UI Framework | WPF |
| Language | C# / .NET 8 |
| Database | SQLite |
| Audio Engine | NAudio |
| Deployment | Self-contained / Framework-dependent |

**Use Case:** Schools, colleges, universities, factories, offices.

---

## ✨ Features

### 🔔 Bell Schedule
- Up to 15 bells per day
- Weekly scheduling (Mon–Sun + Special Day)
- WAV / MP3 support
- Copy schedules across days
- CSV import/export

### 📊 Dashboard
- Live clock
- Next bell countdown
- System health monitoring
- Holiday mode indicator
- Activity feed
- Quick controls (Pause / Ring Now)

### 🕹️ Manual Controls
- Ring Now
- Stop Bell
- Pause / Resume
- Emergency Bell (override system)
- Volume control

### 🏖️ Holiday Management
- Custom holiday dates
- Weekend auto-silencing toggle
- Dashboard holiday alerts

### 📋 Logs
- Activity Log (all users)
- Audit Log (Owner only)
- CSV export support

### 🔐 User Management
| Role | Access |
|------|--------|
| Owner | Full control |
| Viewer | Read-only |

### ⚙️ Settings
- Startup launch toggle
- Auto shutdown option
- Backup / restore database
- Reset system (danger zone)

### 🌙 System Behavior
- Dark / Light mode
- System tray integration
- Background scheduling

## 📖 Tutorial

### Installation
1. Download release
2. Install / run executable
3. (Lite only) Install .NET 8 runtime
4. Launch application

---

### First Launch
- Enter School Name
- Create Owner account
- Login required on next startup

---

### Setting Bell Schedule
1. Select day
2. Set time (HH:mm)
3. Choose audio file
4. Save schedule

---

### Holidays
- Add date (YYYY-MM-DD)
- Add description
- Enable weekend auto-silence if needed

---

### Manual Controls
- Ring instantly
- Stop active bell
- Pause system
- Emergency override
- Adjust volume

---

### Users
- Add users (Owner only)
- Assign roles
- Manage access levels

---

### Logs
- View activity history
- Track system events
- Export logs

---

### Backup
- Export full database
- Restore from backup file
- Reset system if needed

---

### Troubleshooting

| Issue | Fix |
|------|-----|
| Bells not ringing | Check pause mode + file paths |
| Startup issue | Enable startup setting |
| Wrong time | Check system clock |
| Missing sound | Update file path |
| Database error | Restore backup |
